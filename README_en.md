# network-toolbox

A network toolset that provides various networking utilities.

## Component List

| Component | Description | Features |
|-----------|-------------|----------|
| **share-note** | WebSocket-based real-time shared editing tool | Real-time sync, responsive design, auto-reconnect |
| **curl** | Multi-protocol HTTP client tool | HTTP/1.1/2/3 support, certificate verification, POST requests, delayed sending |
| **http-echo** | Simple HTTP echo service | Echo mode, random string generation, speed limiting |
